PARAGRAPH

HTML paragraphs are used to structure and display content as separate blocks on a web page. They provide a way to separate and organize textual information, making it more readable and visually appealing.

To create an HTML paragraph, you use the <p> element. It consists of an opening tag <p> and a closing tag </p> to define the beginning and end of the paragraph.

Text enclosed within the <p> element is displayed as a separate paragraph. By default, there is some vertical space (margin) above and below the paragraph, visually separating it from surrounding content.

To insert line breaks within a paragraph without starting a new paragraph, you can use the <br> tag. It creates a new line within the same paragraph.
