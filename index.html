ATTRIBUTES

HTML attributes are additional pieces of information that you can include within HTML tags to modify the behavior or appearance of HTML elements. Attributes provide metadata about the elements and help define how those elements should be displayed or interact with other elements or scripts. HTML attributes are always specified in the start tag and are typically presented as name-value pairs.

PARAGRAPH

In HTML, a "paragraph" is defined using the '<p>' element, which represents a block of text or content separated from other content, typically with some vertical space above and below it. It is a fundamental HTML element for structuring and displaying text on a web page.

STYLES

Style are the rules and characteristics that are used to define the presentation and look of elements on a web page, as well as to add styles to an element, such as color, font, size, and others. HTML is generally responsible for structuring a web page's content, whereas styles determine how that content is displayed.
